# MVP User Flow Priority

## P0 flows

1. Chrome bare horizontal wheel remains Logi Options+ owned.
2. Chrome Option + horizontal wheel zooms page.
3. Non-Chrome pass-through.
4. Disable restores pass-through.
5. Quit restores system behavior.
6. Uninstall removes app/config/logs and leaves no running process.

## P1 flows

1. Restart app after permission grant.
2. Dry Run mode logs decisions without changing behavior.
3. Missing permissions state is understandable.
4. User can find logs.

## P2 flows

1. Start at login.
2. Signed/notarized public release.
3. PKG package in addition to DMG.
