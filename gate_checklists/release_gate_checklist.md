# Release Gate Checklist

Do not start public release until human acceptance is complete.

## Required before release

- [ ] Human accepted RC exists.
- [ ] Release strategy decided.
- [ ] DMG/PKG choice decided.
- [ ] Signing/notarization choice decided.
- [ ] Changelog drafted.
- [ ] Release notes drafted.
- [ ] Rollback/uninstall docs complete.
- [ ] GitHub Release workflow tested.

## For public macOS release

- [ ] Developer ID signing configured.
- [ ] Notarization configured.
- [ ] Gatekeeper install tested on clean machine/user.
- [ ] SHA256 published.
