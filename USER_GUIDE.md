# User Guide

This is a scaffold. The final app should behave as follows.

## Expected MVP behavior

In Chrome:

- Bare horizontal wheel: unchanged; Logi Options+ can keep handling it.
- Option + horizontal wheel: page zoom.
- Control + horizontal wheel: switch to the previous or next tab.

Outside Chrome:

- ChromeWheelRouter does nothing.

## Expected controls

The final menu bar app should include:

- Enable / Disable
- Dry Run
- Open Permission Settings
- Open Logs
- Quit

## Permissions

The final app will likely need macOS Input Monitoring and Accessibility permissions. These must be granted manually by the user.
